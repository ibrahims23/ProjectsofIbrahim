1)Identifying Sales Amount Project Using MySql and PowerBI

This project is about a hardware manufacturing company known as Atliq Hardware. I was given a raw file of unstructured data ,having duplicate values along with numerous inconsistencies. I have used the MySQL Workbench graphical user interface (GUI) tool for managing and developing databases using the MySQL database management system. This made me accomplish my data preprocessing tasks such as Data Cleaning and Data Merging. After Data Cleaning i connected MySQL WorkBench with PowerBI using the sql-connector. So i could retrieve data in Power BI and my choice was Power BI because it was more convinient as it also allows businesses to create interactive dashboards and reports that provide a visual representation of their data. This makes it easier to understand and analyze business information, allowing for better decision-making.
